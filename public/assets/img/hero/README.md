# Hero imagery

Drop a single high-fidelity photo here as `skyline.jpg` (or adjust the path
in `src/sections/Hero.jsx`). Recommended:

- 2400 × 1600 minimum, WebP or AVIF preferred.
- Tonal range that flatters the navy + champagne-gold overlay — dusk skylines,
  private-aviation tarmac, Mediterranean coastline, quiet interior portraits.
- Avoid faces, logos and any licensed content you cannot clear.

If the file is missing the hero falls back to the CSS navy/gold gradient and
still reads as premium.
